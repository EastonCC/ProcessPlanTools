# ProcessPlanTools

A web-based toolbox to help myself and others working in ProcessPlan

## Website

Visit the live site at: [https://eastoncc.github.io/ProcessPlanTools/](https://eastoncc.github.io/ProcessPlanTools/)

## Tools

- **Formula Editor**: Quickly and neatly write up and edit formulas. They will be formatted nicely with syntax coloring and can be copied with or without the formatting. 

## Development

This is a static site built with vanilla HTML, CSS, and JavaScript.
